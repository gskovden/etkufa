Текущие проекты

Разработка синхронного электродвигателя для электропривода реверсивного устройства

Целью работ является определение конструктивного облика синхронного электродвигателя, основных требований к его рабочим характеристикам, определение его массогабаритных характеристик, а также создание опытного образца синхронного электродвигателя. Электродвигатель должен представлять собой неявнополюсную синхронную машину с возбуждением постоянными магнитами, установленными на её роторе.

Разработка стартер-генератора для турбовинтового двигателя ТВ7-117СТ-01 и его модификаций

Целью работы является разработка стартер-генератора для турбовинтового двигателя ТВ7-117СТ-01 и его модификаций. Стартер-генератор предназначен для раскрутки ротора турбокомпрессора двигателя в процессе запуска, холодной прокрутки, ложного запуска и консервации и генерирования электроэнергии для питания потребителей бортовой сети постоянного тока самолета. Объектом запуска является авиационный турбовинтовой двигатель со свободной турбиной ТВ7-117СТ-01, входящий в состав силовой установки самолета Ил-114-300.

Разработка электромагнитной тормозной муфты для электропривода реверсивного устройства

Целью работы является определение конструктивного облика электромагнитной тормозной муфты, основных требований к его рабочим характеристикам, определение его массогабаритных характеристик.

Изготовление экспериментального образца стартер-генератора и системы управления с уточненными параметрами

Целью работы является оптимизация массо–габаритных характеристик стартер–генератора и изготовление экспериментального образца стартер–генератора с уточненными параметрами

Разработка электрического двигателя с системой управления для маршевой ГСУ лёгкого вертолета

Целью работы являются расчётное–аналитические исследования электродвигателей с различными вариантами исполнения. В процессе работы выполняются расчётно-параметрические исследования по формированию облика электродвигателя с системой управления для маршевой ГСУ легкого вертолета, формирование облика отказоустойчивой системы управления электрическим двигателем, разработка ТЗ на электродвигатель с системой управления для маршевой ГСУ легкого вертолета, разработка конструкторской документации на изготовление ДСЕ электродвигателя и его системы управления для маршевой ГСУ легкого вертолета, изготовление ДСЕ электродвигателя для маршевой ГСУ легкого вертолета, изготовление системы управления электродвигателя для маршевой ГСУ легкого вертолета.

Расчетно-экспериментальные исследования макета синхронной электрической трансмиссии ГСУ

Целью работы являются расчетно-экспериментальные исследования узлов и систем ГСУ с использованием и без использования ВТСП, бортовых систем региональных самолетов. В процессе выполнения работы выполняется анализ технического уровня совершенства технологий, необходимых для создания синхронной электрической трансмиссии ГСУ, разработка математической модели расчета характеристик синхронной электрической трансмиссии ГСУ, экспериментальные исследования макета синхронной электрической трансмиссии ГСУ мощностью до 10 кВт, расчетные исследования синхронной электрической трансмиссии ГСУ мощностью не менее 800 кВт.

Разработка и изготовление ДСЕ, в том числе блоков силовой электроники для электрических машин демонстраторов технологий ГСУ с частично и полностью сверхпроводящим электроэнергетическим комплексом

Целью работы являются разработка и изготовление ДСЕ, в том числе блоков силовой электроники для электрических машин демонстраторов технологий ГСУ с частично и полностью сверхпроводящим электроэнергетическим комплексом . В процессе выполнения работы выполняется проведение аналитических расчётов и выбор оптимальных конструктивных схем электрических машин демонстраторов технологий ГСУ с частично и полностью сверхпроводящим электроэнергетическим комплексом, разработка КД на электрические машины демонстраторов технологий ГСУ с частично и полностью сверхпроводящим электроэнергетическим комплексом, изготовление деталей сборочных единиц электрических машин демонстраторов технологий ГСУ с частично и полностью сверхпроводящим электроэнергетическим комплексом
